MainControllerApi

인터페이스 기반으로 추출한 엔드포인트 문서입니다.

메인 페이지 조회 GET /main
GET

메인 페이지 조회

비로그인 사용자를 포함한 모든 방문자가 메인 페이지에서 QNA, INFO, FREE, NOTICE, EVENT 최신 게시글과 인기글을 한 번에 조회합니다.

GET/main

응답 바디 필드

필드명 타입 필수 여부 설명
event List<PostSummaryResponse> optional EVENT 게시판의 최신 5개 게시글 요약 목록입니다.
free List<PostSummaryResponse> optional FREE 게시판의 최신 5개 게시글 요약 목록입니다.
hotPosts List<HotPostResponse> optional 최근 7일 기준 반응 점수가 높은 인기글 5개 목록입니다.
info List<PostSummaryResponse> optional INFO 게시판의 최신 5개 게시글 요약 목록입니다.
notice List<PostSummaryResponse> optional NOTICE 게시판의 최신 5개 게시글 요약 목록입니다.
qna List<PostSummaryResponse> optional QNA 게시판의 최신 5개 게시글 요약 목록입니다.
