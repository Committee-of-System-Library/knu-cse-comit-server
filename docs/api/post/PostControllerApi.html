POST

게시글 작성

새 게시글을 작성합니다.

POST/posts

요청 바디 필드

필드명 타입 필수 여부 설명
boardType BoardType required 게시글을 작성할 게시판 유형입니다.
content String required 게시글 본문입니다. 최대 500자입니다.
tags List<String> optional 선택 항목입니다. 최대 5개, 각 20자 이하입니다.
title String required 게시글 제목입니다. 최대 30자입니다.

에러 응답 필드

필드명 타입 필수 여부 설명
type String required 문제 유형을 식별하는 URI path입니다.
title String required HTTP 상태를 요약한 제목입니다.
status Integer required HTTP 상태 코드입니다.
detail String required 클라이언트에 노출할 에러 설명입니다.
instance String required 에러가 발생한 요청 경로입니다.
errorCode String required 프론트엔드 분기에 사용하는 안정적인 에러 코드입니다.
invalidFields List<ProblemFieldViolation> optional validation 에러에서만 포함됩니다. 필드명과 검증 메시지 목록입니다.
errorTrackingId String optional 서버 내부 오류에서만 포함됩니다. 로그 추적용 식별자입니다.
timestamp OffsetDateTime required 에러 응답 생성 시각입니다.

에러 코드

errorCode HTTP 상태 type detail 발생 조건
UNAUTHORIZED 401 /problems/common/unauthorized 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
INVALID_REQUEST 400 /problems/common/invalid-request 입력값이 올바르지 않습니다. 요청 파라미터 또는 본문이 검증 규칙을 만족하지 않을 때
MEMBER_NOT_FOUND 404 /problems/member/not-found 존재하지 않는 회원입니다. 인증된 사용자의 로컬 회원 정보가 존재하지 않을 때
INVALID_TAG 400 /problems/post/invalid-tag 태그는 1~20자이어야 합니다. 태그 길이가 도메인 규칙을 벗어날 때

요청 예시

{
  "boardType" : "QNA",
  "title" : "JPA fetch join 질문",
  "content" : "join fetch와 entity graph 차이가 궁금합니다.",
  "tags" : [ "spring", "jpa" ]
}

응답 예시

{
  "result" : "SUCCESS",
  "data" : 101
}

에러 예시

{
  "type" : "/problems/common/unauthorized",
  "title" : "Unauthorized",
  "status" : 401,
  "detail" : "인증이 필요합니다.",
  "instance" : "/posts",
  "errorCode" : "UNAUTHORIZED",
  "timestamp" : "2026-03-24T03:10:00Z"
}

GET

게시글 목록 조회

게시판별 게시글 목록을 cursor 기반으로 조회합니다.

GET/posts

쿼리 파라미터

필드명 타입 필수 여부 설명
boardType BoardType required 조회할 게시판 유형입니다. QNA 또는 FREE를 사용합니다.
cursor Long optional 이전 응답의 nextCursorId. 첫 페이지는 생략합니다.
size int optional 조회할 게시글 수입니다. 기본값은 20이고 최대 20입니다.

응답 바디 필드

필드명 타입 필수 여부 설명
hasNext boolean optional 다음 페이지 존재 여부입니다.
nextCursorId Long optional 다음 페이지 조회에 사용할 cursor ID입니다. 마지막 페이지면 null입니다.
posts List<PostSummaryResponse> optional 게시글 요약 목록입니다. 각 항목은 게시글 ID, 제목, 작성자, 좋아요 수, 댓글 수, 태그, 작성 시각을 포함합니다.
