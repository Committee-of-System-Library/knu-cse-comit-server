POST

공지/이벤트/정보 게시글 작성 (관리자)

관리자가 NOTICE, EVENT, INFO 게시판에 게시글을 작성합니다. QNA, FREE 게시판에는 작성할 수 없습니다.

POST/admin/posts

요청 바디 필드

필드명 타입 필수 여부 설명
boardType BoardType required 게시판 유형입니다. NOTICE, EVENT, INFO 중 하나만 허용됩니다.
content String required 게시글 내용입니다.
imageUrls List<String> optional 첨부 이미지 URL 목록입니다. 생략 가능합니다.
tags List<String> optional 태그 목록입니다. 생략 가능합니다.
title String required 게시글 제목입니다.

응답 바디 필드

필드명 타입 필수 여부 설명
postId Long optional 생성된 게시글 ID입니다.

에러 응답 필드

필드명 타입 필수 여부 설명
type String required 문제 유형을 식별하는 URI path입니다.
title String required HTTP 상태를 요약한 제목입니다.
status Integer required HTTP 상태 코드입니다.
detail String required 클라이언트에 노출할 에러 설명입니다.
instance String required 에러가 발생한 요청 경로입니다.
errorCode String required 프론트엔드 분기에 사용하는 안정적인 에러 코드입니다.
invalidFields List<ProblemFieldViolation> optional validation 에러에서만 포함됩니다. 필드명과 검증 메시지 목록입니다.
errorTrackingId String optional 서버 내부 오류에서만 포함됩니다. 로그 추적용 식별자입니다.
timestamp OffsetDateTime required 에러 응답 생성 시각입니다.

에러 코드

errorCode HTTP 상태 type detail 발생 조건
UNAUTHORIZED 401 /problems/common/unauthorized 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
INVALID_REQUEST 400 /problems/common/invalid-request 입력값이 올바르지 않습니다. 요청 본문이 검증 규칙을 만족하지 않을 때
FORBIDDEN 403 /problems/common/forbidden 접근 권한이 없습니다. 관리자 권한이 없는 사용자가 요청할 때
FORBIDDEN_BOARD_TYPE 400 /problems/post/forbidden-board-type 해당 게시판 유형은 관리자만 작성할 수 있습니다. NOTICE, EVENT, INFO 외 boardType으로 요청할 때

요청 예시

{
  "boardType" : "NOTICE",
  "title" : "2026년 1학기 정기 모집 공고",
  "content" : "모집 내용입니다.",
  "tags" : [ "모집", "공지" ],
  "imageUrls" : [ ]
}

응답 예시

{
  "result" : "SUCCESS",
  "data" : {
    "postId" : 42
  }
}

에러 예시

{
  "type" : "/problems/common/unauthorized",
  "title" : "Unauthorized",
  "status" : 401,
  "detail" : "인증이 필요합니다.",
  "instance" : "/admin/posts",
  "errorCode" : "UNAUTHORIZED",
  "timestamp" : "2026-03-24T03:10:00Z"
}
