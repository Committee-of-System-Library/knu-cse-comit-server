AdminReportControllerApi

인터페이스 기반으로 추출한 엔드포인트 문서입니다.

신고 상세 조회 GET /admin/reports/{reportId} 신고 목록 조회 GET /admin/reports 신고 상태 변경 PATCH /admin/reports/{reportId}/status
GET

신고 상세 조회

관리자가 신고 상세 정보를 조회합니다.

GET/admin/reports/{reportId}

경로 변수

필드명 타입 필수 여부 설명
reportId Long required 조회할 신고 ID입니다.

응답 바디 필드

필드명 타입 필수 여부 설명
createdAt LocalDateTime optional 신고 생성 시각입니다.
id Long optional 신고 ID입니다.
message String optional 신고 사유 메시지입니다.
reporterNickname String optional 신고자 닉네임입니다.
reviewedAt LocalDateTime optional 처리 시각입니다. 미처리 상태이면 null입니다.
reviewedByNickname String optional 처리한 관리자 닉네임입니다. 미처리 상태이면 null입니다.
status ReportStatus optional 신고 상태입니다.
targetId Long optional 신고 대상 ID입니다.
targetType ReportTargetType optional 신고 대상 유형입니다.

에러 응답 필드

필드명 타입 필수 여부 설명
type String required 문제 유형을 식별하는 URI path입니다.
title String required HTTP 상태를 요약한 제목입니다.
status Integer required HTTP 상태 코드입니다.
detail String required 클라이언트에 노출할 에러 설명입니다.
instance String required 에러가 발생한 요청 경로입니다.
errorCode String required 프론트엔드 분기에 사용하는 안정적인 에러 코드입니다.
invalidFields List<ProblemFieldViolation> optional validation 에러에서만 포함됩니다. 필드명과 검증 메시지 목록입니다.
errorTrackingId String optional 서버 내부 오류에서만 포함됩니다. 로그 추적용 식별자입니다.
timestamp OffsetDateTime required 에러 응답 생성 시각입니다.

에러 코드

errorCode HTTP 상태 type detail 발생 조건
UNAUTHORIZED 401 /problems/common/unauthorized 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
FORBIDDEN 403 /problems/common/forbidden 접근 권한이 없습니다. 관리자 권한이 없는 사용자가 요청할 때
REPORT_NOT_FOUND 404 /problems/report/not-found 존재하지 않는 신고입니다. 존재하지 않는 신고 ID로 요청할 때

응답 예시

{
  "result" : "SUCCESS",
  "data" : {
    "id" : 1,
    "targetType" : "POST",
    "targetId" : 10,
    "message" : "광고성 도배입니다",
    "reporterNickname" : "reporter-1",
    "status" : "REVIEWED",
    "createdAt" : "2026-03-28T10:00:00",
    "reviewedAt" : "2026-03-28T11:00:00",
    "reviewedByNickname" : "admin-1"
  }
}

에러 예시

{
  "type" : "/problems/common/unauthorized",
  "title" : "Unauthorized",
  "status" : 401,
  "detail" : "인증이 필요합니다.",
  "instance" : "/admin/reports/{reportId}",
  "errorCode" : "UNAUTHORIZED",
  "timestamp" : "2026-03-24T03:10:00Z"
}
GET

신고 목록 조회

관리자가 접수된 신고 목록을 필터링하여 조회합니다. status와 targetType은 선택 필터입니다.

GET/admin/reports

쿼리 파라미터

필드명 타입 필수 여부 설명
status ReportStatus optional 신고 상태 필터입니다. RECEIVED, REVIEWED, DISMISSED, ACTIONED 중 하나를 사용합니다. 생략하면 전체를 조회합니다.
targetType ReportTargetType optional 신고 대상 유형 필터입니다. POST 또는 COMMENT를 사용합니다. 생략하면 전체를 조회합니다.

응답 바디 필드

필드명 타입 필수 여부 설명
page int optional 현재 페이지 번호입니다. 0부터 시작합니다.
reports List<AdminReportSummaryResponse> optional 신고 요약 목록입니다.
size int optional 페이지 크기입니다.
totalElements long optional 전체 신고 수입니다.
totalPages int optional 전체 페이지 수입니다.

PATCH

신고 상태 변경

관리자가 접수된 신고의 상태를 변경합니다. RECEIVED 상태의 신고만 변경할 수 있습니다.

PATCH/admin/reports/{reportId}/status

경로 변수

필드명 타입 필수 여부 설명
reportId Long required 상태를 변경할 신고 ID입니다.

요청 바디 필드

필드명 타입 필수 여부 설명
status ReportStatus required 변경할 상태입니다. REVIEWED, DISMISSED, ACTIONED 중 하나를 사용합니다.

에러 응답 필드

필드명 타입 필수 여부 설명

detail String required 클라이언트에 노출할 에러 설명입니다.
instance String required 에러가 발생한 요청 경로입니다.
errorCode String required 프론트엔드 분기에 사용하는 안정적인 에러 코드입니다.
invalidFields List<ProblemFieldViolation> optional validation 에러에서만 포함됩니다. 필드명과 검증 메시지 목록입니다.
errorTrackingId String optional 서버 내부 오류에서만 포함됩니다. 로그 추적용 식별자입니다.

에러 코드

INVALID_REQUEST 400 /problems/common/invalid-request 입력값이 올바르지 않습니다. 변경 상태가 REVIEWED, DISMISSED, ACTIONED 중 하나가 아닐 때
