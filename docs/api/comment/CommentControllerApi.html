CommentControllerApi

인터페이스 기반으로 추출한 엔드포인트 문서입니다.

댓글 작성 POST /posts/{postId}/comments 댓글 삭제 DELETE /comments/{commentId} 댓글 목록 조회 GET /posts/{postId}/comments 댓글 신고 POST /comments/{commentId}/reports 댓글 도움 토글 POST /comments/{commentId}/helpful 댓글 수정 PATCH /comments/{commentId}
POST

댓글 작성

게시글에 새 댓글 또는 대댓글을 작성합니다. parentCommentId가 없으면 일반 댓글, 있으면 해당 댓글의 대댓글로 생성됩니다.

POST/posts/{postId}/comments

경로 변수

필드명 타입 필수 여부 설명
postId Long required 댓글을 작성할 게시글 ID

요청 바디 필드

필드명 타입 필수 여부 설명
content String required 댓글 본문입니다. 비어 있을 수 없습니다.
parentCommentId Long optional 선택 항목입니다. 대댓글 대상 최상위 댓글 ID입니다. 대댓글의 대댓글은 허용하지 않습니다.

에러 응답 필드

필드명 타입 필수 여부 설명
type String required 문제 유형을 식별하는 URI path입니다.
title String required HTTP 상태를 요약한 제목입니다.
status Integer required HTTP 상태 코드입니다.
detail String required 클라이언트에 노출할 에러 설명입니다.
instance String required 에러가 발생한 요청 경로입니다.
errorCode String required 프론트엔드 분기에 사용하는 안정적인 에러 코드입니다.
invalidFields List<ProblemFieldViolation> optional validation 에러에서만 포함됩니다. 필드명과 검증 메시지 목록입니다.
errorTrackingId String optional 서버 내부 오류에서만 포함됩니다. 로그 추적용 식별자입니다.
timestamp OffsetDateTime required 에러 응답 생성 시각입니다.

에러 코드

errorCode HTTP 상태 type detail 발생 조건
UNAUTHORIZED 401 /problems/common/unauthorized 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
INVALID_REQUEST 400 /problems/common/invalid-request 입력값이 올바르지 않습니다. 요청 파라미터 또는 본문이 검증 규칙을 만족하지 않을 때
POST_NOT_FOUND 404 /problems/post/not-found 존재하지 않는 게시글입니다. 댓글을 작성할 게시글이 없거나 삭제된 상태일 때
COMMENT_NOT_FOUND 404 /problems/comment/not-found 존재하지 않는 댓글입니다. parentCommentId로 지정한 댓글이 없거나 삭제된 상태일 때
INVALID_PARENT_COMMENT 400 /problems/comment/invalid-parent 유효한 부모 댓글이 아닙니다. 다른 게시글의 댓글이거나 이미 대댓글인 댓글에 답글을 달려고 할 때
MEMBER_NOT_FOUND 404 /problems/member/not-found 존재하지 않는 회원입니다. 인증된 사용자의 로컬 회원 정보가 존재하지 않을 때

요청 예시

{
  "parentCommentId" : 201,
  "content" : "entity graph도 같이 비교해보면 좋습니다."
}

응답 예시

{
  "result" : "SUCCESS",
  "data" : 201
}

에러 예시

{
  "type" : "/problems/common/unauthorized",
  "title" : "Unauthorized",
  "status" : 401,
  "detail" : "인증이 필요합니다.",
  "instance" : "/posts/{postId}/comments",
  "errorCode" : "UNAUTHORIZED",
  "timestamp" : "2026-03-24T03:10:00Z"
}

댓글 신고

활성 댓글을 신고합니다. 신고 사유는 자유 입력 메시지이며 최대 500자까지 허용합니다.

POST/comments/{commentId}/reports

경로 변수

필드명 타입 필수 여부 설명

message String required 신고 사유 메시지입니다. 공백만 입력할 수 없고 최대 500자까지 허용합니다.

응답 바디 필드

필드명 타입 필수 여부 설명
reportId Long optional 생성된 신고 ID입니다.
