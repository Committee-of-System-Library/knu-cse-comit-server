CommentControllerApi

인터페이스 기반으로 추출한 엔드포인트 문서입니다.

댓글 작성 POST /posts/{postId}/comments 댓글 삭제 DELETE /comments/{commentId} 댓글 목록 조회 GET /posts/{postId}/comments 댓글 도움 토글 POST /comments/{commentId}/helpful 댓글 수정 PATCH /comments/{commentId}
POST

댓글 작성

게시글에 새 댓글을 작성합니다.

POST/posts/{postId}/comments

경로 변수

필드명 타입 필수 여부 설명
postId Long required 댓글을 작성할 게시글 ID

요청 바디 필드

필드명 타입 필수 여부 설명
content String required 댓글 본문

에러 응답 필드

필드명 타입 필수 여부 설명
result String required 요청 처리 결과입니다. 에러 응답에서는 FAIL이 내려갑니다.
code String required 비즈니스 에러 코드입니다.
message String required 에러 메시지입니다.

에러 코드

이름 HTTP 상태 응답 code 메시지 발생 조건
UNAUTHORIZED 401 C001 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
POST_NOT_FOUND 404 P001 존재하지 않는 게시글입니다. 댓글을 작성할 게시글이 없거나 삭제된 상태일 때
MEMBER_NOT_FOUND 404 M001 존재하지 않는 회원입니다. 인증된 사용자의 로컬 회원 정보가 존재하지 않을 때

요청 예시

{
  "content" : "entity graph도 같이 비교해보면 좋습니다."
}

응답 예시

{
  "result" : "SUCCESS",
  "data" : 201
}

에러 예시

{
  "result" : "FAIL",
  "code" : "C001",
  "message" : "인증이 필요합니다."
}
DELETE

댓글 삭제

내가 작성한 댓글을 삭제 상태로 변경합니다.

DELETE/comments/{commentId}

경로 변수

필드명 타입 필수 여부 설명
commentId Long required 삭제할 댓글 ID

에러 응답 필드

필드명 타입 필수 여부 설명
result String required 요청 처리 결과입니다. 에러 응답에서는 FAIL이 내려갑니다.
code String required 비즈니스 에러 코드입니다.
message String required 에러 메시지입니다.

에러 코드

이름 HTTP 상태 응답 code 메시지 발생 조건
UNAUTHORIZED 401 C001 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
COMMENT_NOT_FOUND 404 CM001 존재하지 않는 댓글입니다. 삭제 대상 댓글이 없거나 삭제된 상태일 때

응답 예시

}

에러 예시

  "code" : "C001",
  "message" : "인증이 필요합니다."

GET

댓글 목록 조회

게시글에 달린 댓글 목록을 조회합니다.

GET/posts/{postId}/comments

경로 변수

필드명 타입 필수 여부 설명
postId Long required 댓글을 조회할 게시글 ID

응답 바디 필드

필드명 타입 필수 여부 설명
comments List<CommentResponse> optional 댓글 목록입니다. 각 항목은 댓글 ID, 내용, 작성자, 도움이 됐어요 수, 내 반응 여부, 작성/수정 시각을 포함합니다.

에러 응답 필드

필드명 타입 필수 여부 설명
result String required 요청 처리 결과입니다. 에러 응답에서는 FAIL이 내려갑니다.
code String required 비즈니스 에러 코드입니다.
message String required 에러 메시지입니다.

에러 코드

이름 HTTP 상태 응답 code 메시지 발생 조건
UNAUTHORIZED 401 C001 인증이 필요합니다. 인증 정보가 없거나 유효한 회원을 식별할 수 없을 때
POST_NOT_FOUND 404 P001 존재하지 않는 게시글입니다. 댓글을 조회할 게시글이 없거나 삭제된 상태일 때

응답 예시

{
  "result" : "SUCCESS",
  "data" : {
    "comments" : [ {
      "id" : 201,
      "content" : "entity graph도 같이 비교해보면 좋습니다.",
      "authorNickname" : "orm-master",
      "helpfulCount" : 4,
      "helpfulByMe" : true,
      "mine" : false,
      "createdAt" : "2026-03-24T11:00:00",
      "updatedAt" : null
    } ]
  }
}

에러 예시

{
  "result" : "FAIL",
  "code" : "C001",
  "message" : "인증이 필요합니다."
}

COMMENT_NOT_FOUND 404 CM001 존재하지 않는 댓글입니다. 도움이 됐어요를 누를 댓글이 없거나 삭제된 상태일 때
