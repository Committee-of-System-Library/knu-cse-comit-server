PaymentControllerApi

인터페이스 기반으로 추출한 엔드포인트 문서입니다.

결제 승인 POST /v1/payments/confirm 결제 조회 GET /v1/payments/{orderId}
POST

결제 승인

결제 승인 요청을 처리하고 승인 결과를 반환합니다.

POST/v1/payments/confirm

요청 바디 필드

필드명 타입 필수 여부 설명
amount Integer required 결제 금액 (원 단위)
orderId String required 주문 ID
paymentKey String required 토스 결제 고유 키

응답 바디 필드

필드명 타입 필수 여부 설명
amount Integer optional 결제 금액 (원 단위)
approvedAt OffsetDateTime optional 결제 승인 시각
orderId String optional 주문 ID
status String optional 결제 상태

요청 예시

{
  "orderId" : "ORDER-001",
  "amount" : 15000,
  "paymentKey" : "pay_123456789"
}

응답 예시

{
  "status" : "DONE",
  "approvedAt" : "2024-01-01T12:00:00Z",
  "orderId" : "ORDER-001",
  "amount" : 15000
}
GET

결제 조회

주문 ID 기준으로 결제 상태를 조회합니다.

GET/v1/payments/{orderId}

경로 변수

필드명 타입 필수 여부 설명
orderId String required 조회할 주문 ID

쿼리 파라미터

필드명 타입 필수 여부 설명
includeHistory boolean optional 거래 이력 포함 여부

응답 바디 필드

필드명 타입 필수 여부 설명
approvedAt OffsetDateTime optional 결제 승인 시각
historyIncluded boolean optional 거래 이력 포함 여부
orderId String optional 조회할 주문 ID
status String optional 결제 상태

응답 예시

{
  "orderId" : "ORDER-001",
  "status" : "DONE",
  "approvedAt" : "2024-01-01T12:00:00Z",
  "historyIncluded" : true
}
